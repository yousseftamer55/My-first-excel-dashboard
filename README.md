# Employee Salary Dashboard (Excel)

## 📊 Overview
This project is an interactive Excel dashboard analyzing employee salaries, departments, and workforce distribution.

## 🛠 Tools Used
- Microsoft Excel

## 📈 Features
- Salary analysis by gender, department, and position
- Employee distribution across branches
- Hiring trends over time
- Interactive filters

## ⚠️ Note
The original dataset is not available, but the dashboard demonstrates data visualization and reporting skills.
